# ReedIsland✨

一个面向芦苇岛（aweidao.com）的安卓客户端，在[DawnIslandK](https://github.com/fishballzzz/DawnIslandK)的基础上开发得来。

## Contribute

这是一个开源项目，你可以在符合[DawnIslandK](https://github.com/fishballzzz/DawnIslandK)版权许可的前提下自由进行修改和贡献。
